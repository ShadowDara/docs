# Ansicolors

[Back Home](#) - [Download Page](#/download) - [Search](#/search)

This is a tiny header only lib for C++ and C which add every ansi color as a macro. Just copy the header file to your project.

If you are using raylib add this macro because otherwise the macros will be overwriteen by raylib.

```c++
#define RM_ANSI_FROM_COLORNAME 1

```

You can use it as a const too by defining:

```c++
#define USE_COLORS_AS_MACRO 1
```

The repo can be found [here](https://github.com/ShadowDara/colors-cpp).
